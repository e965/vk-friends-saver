Сохранение списка друзей пользователя ВК

Внимание! Согласно правилам использования API ВКонтакте, вам необходимо пройти авторизацию, иначе приложение работать не будет.
Сервис не собирает и не хранит никаких персональных данных пользователей. Токен авторизации сохраняется в браузере пользователя приложения, а все запросы выполняются напрямую к API ВКонтакте. Ознакомьтесь с политикой конфиденциальности ВК.

Введите цифры, которые идут после /id в адресе. Например, для того, чтобы получить список друзей у vk.com/id1, нужно ввести 1. Узнать полный id пользователя, если у него поставлен ник (напр. vk.com/durov) можно здесь. Скрытых друзей можно попробовать поискать тут.

Если при запросе выдаётся ошибка Error: JSONP request to https://api.vk.com/..., то необходимо отключить блокировщики рекламы и/или скриптов.
